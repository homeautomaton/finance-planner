Instructions

Graph: shows Balance, Salary, and Spending over time.

Left panel: controls base assumptions. Changes here affect the whole forecast.

JSON box: full state representation. Can be edited directly or generated from interactions.

Apply: runs the JSON in the box and updates the simulation state.

Download: save JSON to your local device.

Upload: load JSON from your local device.

Dragging points: overrides a single year’s value and updates JSON automatically, normally snapped to nearest 100.

Shift + drag: override without snap to nearest 100

Opt/Alt-click: removes a saved override at a point.

Saved points: visually marked (cross / hollow style).
